How To Setup

Follow these steps to set up IGI 2 multiplayer:

  • Download the game installer from the Downloads page.
  • Run the installer and follow the on-screen instructions.
  • Configure the game settings as per your preference.
  • Join our community for additional help and tips.
  • Joins The Servers and Enjoy multiplayer game.
